# Amazon Sales Analytics Project

## Overview

This project analyzes Amazon marketplace product data to uncover insights related to pricing strategy, customer satisfaction, and customer demand across categories.

Using Python-based data analysis, the project explores how discounts influence ratings, which categories dominate engagement, and where sellers may find growth opportunities.

---

## Objectives

The main goals of this project were to:

- Analyze pricing trends across product categories  
- Compare discounts and premium pricing segments  
- Evaluate product quality using ratings and review counts  
- Identify high-demand categories and subcategories  
- Generate actionable business recommendations

---

## Dataset Source

Kaggle Amazon Sales Dataset

The dataset includes:

- Product Name  
- Category  
- Discounted Price  
- Actual Price  
- Discount Percentage  
- Rating  
- Rating Count  
- Product Descriptions
